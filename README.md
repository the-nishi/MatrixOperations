📊 Matrix Operations JavaFX Application

A desktop application built with Java and JavaFX that performs fundamental matrix operations through an interactive graphical user interface. This project demonstrates core concepts of linear algebra combined with GUI-based software development.

🚀 Features
Matrix addition and subtraction
Matrix multiplication
User-friendly JavaFX interface (FXML-based UI)
Event-driven interaction using controllers
Structured project built with NetBeans (Ant-based)
🛠️ Tech Stack
Language: Java
Framework: JavaFX
UI Design: FXML
Build Tool: Apache Ant (NetBeans project)
📂 Project Structure
iZafor/ – Core application classes and controllers
FXMLDocument.fxml – UI layout
App.java – Main application entry point
build/ & dist/ – Compiled outputs and executable JAR
🎯 Purpose

This project was developed as part of coursework to explore:

Matrix manipulation algorithms
GUI development using JavaFX
MVC-style separation with FXML and controllers
